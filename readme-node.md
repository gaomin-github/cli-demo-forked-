# cross-spawn 和child_process 区别？

 1）child_process.spawn() 方法的作用是使用指定的命令行参数创建异步子进程，child_process.spawnSync() 是其同步进程创建方法。

2）cross-spawn 是 node spawn 和 spawnSync 的跨平台实现，使用方式完全一致，主要解决 node spawn 在 Windows 上存在的问题。
